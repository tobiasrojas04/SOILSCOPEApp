# SOILSCOPE
IoT smart irrigation and monitoring system using embedded sensors and real-time data visualization. This project uses MQTT for data transmission.

## Features

- Real-time environmental monitoring
- Sensor data acquisition
- Smart irrigation logic
- Historical data storage
- MQTT communication
- Remote visualization interface
- Embedded firmware development

## Technologies Used

- C++
- TypeScript
- MQTT
- Microcontroller
- Raspberry Pi
- Embedded Systems
- IoT
- Real-Time Monitoring
- Sensors

## Hardware Components

- ESP32
- Soil moisture sensors
- Environmental sensors
- Relay module
- Water pump
- Distance sensor
